# Procedure-Oriented-Curriculum-design

面向过程课程设计 - 设备管理系统

1 - 录入新信息
    * 从曾经保存过的记录（文件）中读入
    * 从键盘（stdin）读入
2 - 查找记录
    * 按编号查找
    * 关键字查找（用strstr()实现）
3 - 修改已有信息
    需要输入编号，修改指定编号的某一项信息
4 - 修改窗口与字体颜色（5/19凌晨完成）
5 - 保存已有记录（到文件）
6 - 退出系统（完成）





要求：
1、输入/添加记录：能够输入、添加多条记录，程序中记录用链表进行存储。
2、显示、查询记录：根据用户要求按所给记录关键字显示、查询一个或多个相关记录。
3、修改记录：能对任意存在的记录进行修改，并能显示修改后的数据。
4、写入文件：程序结束前，把存在的记录写入的磁盘文件中。
